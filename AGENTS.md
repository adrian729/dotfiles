# Dotfiles project — opencode rules

This is a dotfiles repo managed with GNU Stow. The canonical opencode and
Claude Code config lives under `~/.claude/` (symlinked from `claude/.claude/`).

- **Skills**: `~/.claude/skills/` — e.g. audit-loop, local-llm, limit-relay
- **Agents**: `~/.claude/agents/` — e.g. planner, implementer, reviewer
- **Global rules**: `~/.config/opencode/AGENTS.md`
- **Project-level rules**: this file
